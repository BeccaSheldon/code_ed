Spans

This is a span example with only "span example" getting some sweet styling.

Spans allow you to target specific parts of an element without breaking that part away from the rest of the element.

Some elements are known as block elements.
Block elements take up the whole line and push other elements in different directions if they are not also block elements.

Some elements are known as inline elements because. Inline elements do not take up their own line.
